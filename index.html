💾 법제처에서 HWP/PDF/DOC 저장하는 방법

법제처는 보안상 직접 다운로드 링크를 제공하지 않습니다. 다음 절차를 따라주세요:

  1. 이 페이지에서 원하는 법령명을 클릭 → 새 탭에서 법제처 페이지가 열림
  2. 법제처 페이지 우측 상단의 💾 저장 아이콘 클릭
  3. 팝업에서 원하는 형식 선택: HWP / HWPX / PDF / DOC
  4. 인쇄/저장 버튼 클릭 → 파일 다운로드 시작

💡 팁: KITECH 내부규정은 검색 페이지로 연결됩니다. 원문 PDF는 KITECH 정보공개 청구 또는 통합공시 PDF에서 확인하세요.
