# 한국생산기술연구원(KITECH) 관련 법규·규정 리스트

KITECH의 설립근거 법령부터 내부규정까지 카테고리별로 정리한 정적 웹페이지입니다.

🌐 **Live**: https://seunghan91.github.io/kitech-regulations/

## 기능

- 🔍 실시간 검색 (규정명, 비고 텍스트 검색)
- 💾 법제처 다운로드 가이드 모달
- 📋 법령 URL 클립보드 복사
- 🌙 다크모드 (localStorage 저장, 시스템 설정 자동 감지)
- 📱 모바일 반응형
- ⌨️ 키보드 단축키 (`/` 검색 포커스, `Esc` 모달 닫기)

## 데이터 출처

- 법제처 [국가법령정보센터](https://www.law.go.kr) — 상위 법령 원문 링크
- [KITECH 공식 사이트](https://www.kitech.re.kr) — 내부규정 검색 링크
- 2025년 1분기 KITECH 통합공시 PDF — 규정 목록·개정일

## 기술

순수 HTML/CSS/JS 단일 파일. 빌드 도구·프레임워크 없음.
